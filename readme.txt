== Description ==
Official plugin of SalesWizard CRM. Allow Wordpress to send formulas to CRM thank to integration with WP Form and Contact Form 7.

== Installation ==
1. Install Official SalesWizard CRM Plugin either via the WordPress.org plugin repository or by uploading the files to your server. (See instructions on how to install a WordPress plugin)
2. Activate Official SalesWizard CRM Plugin.
3. Navigate to the SalesWizard CRM  tab of your admin menu and click the \"setting tab\"  and fill all the field perfectly



== Frequently Asked Questions ==
= Who should use Official SalesWizard CRM Plugin ? =
-Everybody can use this plugin who need to save wpform field data and contact form 7 data into database. Plugin helps to send data to SalesWizard CRM directly from WordPress site.

= Do I need to have coding skills to use Official SalesWizard CRM Plugin? =
-Absolutely not.
